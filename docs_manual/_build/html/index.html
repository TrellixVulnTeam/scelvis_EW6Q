SCelVis¶

SCelVis is a web-based visualization tool for single-cell sequencing data.

Getting Started¶

  • You can find the publically available web app at https://scelvis-demo.bihealth.org. To get started with your analysis, you can follow the Analysis Tutorial.

  • If you are a bioinformatician and interested in running your own SCelVis server, you can follow the Command Line Tutorial.

A short demonstration of SCelVis in action.¶

Running on your Computer¶

The recommended way to run SCelVis is using Docker:

$ docker run quay.io/biocontainers/scelvis:TAG scelvis --help
$ docker run -p 8050:8050 -v data:/data quay.io/biocontainers/scelvis:TAG scelvis run --data-source /data

In the above, replace TAG with the latest version that you can find on the corresponding Quay.io project. For example, use 0.7.3--py_0.

Alternatively, you can install it using pip…

$ pip install scelvis
$ scelvis run --data-source ./data

… or using Bioconda:

$ conda install scelvis
